## C++:

### 1. 把字符串里面的数字转化为整数：

```c++
#include<iostream>
#include<string>
using namespace std;
int main(){

	int n = 0;
	string str="1234567890";

	for (int i = 0;i < str.length();i++){
		n = str[i] - '0';   //这里把string类型的数字转换为int类型
		cout<<n<<endl;
	}

	return 0;
}
```

运行结果：
```
1
2
3
4
5
6
7
8
9
0
```
